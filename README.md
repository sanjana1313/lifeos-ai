# LifeOS AI

LifeOS AI is a smart student life management system that helps students organize academic and personal tasks and decide what to focus on each day.

## Problem
Students manage assignments, exams, appointments, hobbies, and personal commitments across multiple apps, leading to overload and missed deadlines.

## Solution
LifeOS AI allows students to dump all tasks in one place and then intelligently prioritizes what needs attention today based on deadlines and priority.

## Key Features
- Add academic and personal tasks
- Persistent task storage (no data loss on refresh)
- Daily task prioritization
- Urgent vs non-urgent task filtering
- Simple and beginner-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser Local Storage
- Replit (Deployment)
